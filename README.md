# TestMvc — Web-приложение для управления продуктами

## Описание проекта

TestMvc — это веб-приложение для управления списком продуктов.  
Приложение реализовано на **ASP.NET Core MVC**, использует **Entity Framework Core** и базу данных **Microsoft SQL Server**.  

Приложение позволяет:

- Авторизоваться и управлять продуктами (CRUD: создание, редактирование, просмотр).  
- Получать данные через REST API в формате JSON, включая фильтрацию по названию и диапазону цен.  

Все изменения хранятся в базе данных MSSQL. Авторизация реализована через логин/пароль, указанный в конфигурации приложения.

---

## Реализованный функционал

### Веб-интерфейс

- **Список продуктов** — отображение `id`, `название`, `цена` и ссылка на редактирование.  
- **Создание продукта** — форма для добавления нового продукта.  
- **Редактирование продукта** — форма для изменения существующего продукта.  
- **Авторизация** — доступ к управлению продуктами разрешён только авторизованным пользователям.  

### API

- **GET /api/product** — список продуктов с фильтрацией по `Name`, `MinCost`, `MaxCost`.  
- **GET /api/product/{id}** — детальная информация о продукте по `id`.

- ASP.NET Core MVC

---

## Как это сделано

- **ASP.NET Core MVC** — структура с контроллерами (`ProductController`, `AccountController`), моделями (`Product`, DTO для API) и представлениями (`Views`).  
- **Entity Framework Core** — для работы с MSSQL, миграции и сохранение данных.  
- **Авторизация** — cookie-based, логин и пароль берутся из `appsettings.json`.  
- **API** — отдельные методы контроллера с `[HttpGet]` и `[FromQuery]` для фильтрации.  
- **Docker** — web-приложение и MSSQL запускаются в контейнерах для удобства деплоя.
